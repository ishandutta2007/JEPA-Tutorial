# Contributing to the AI Researcher & Engineer Roadmap

First off, thank you for considering contributing to this project! We welcome any contributions that help us improve and grow these roadmaps.

## How to Contribute

-   **Issues**: If you find a mistake, want to suggest an improvement, or have an idea for a new roadmap, please open an issue.
-   **Pull Requests**: If you want to contribute directly, you can fork the repository, make your changes, and submit a pull request.

## Guidelines

-   Make sure your contributions are in line with the purpose of this repository.
-   Be respectful and follow our [Code of Conduct](./CODE_OF_CONDUCT.md).
-   For any new content, please try to provide references or sources if possible.

We are excited to see your contributions!
